# summarize-me

[![skills.sh](https://skills.sh/b/Aung-IO/summerize-me)](https://skills.sh/Aung-IO/summerize-me)

Daily work-memory summaries for AI-assisted developers.

`summarize-me` helps reconstruct what happened during a workday: what changed, why it changed, which decisions were made, what failed, and what should be continued tomorrow.

## Install

```sh
npx skills add Aung-IO/summerize-me
```

## Command

```sh
summarize-me today
```

The summary is designed to reduce black-box working caused by bouncing between AI prompts, tool calls, commits, notes, and local changes throughout the day.
